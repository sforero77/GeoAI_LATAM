## üí° Estrategias de Optimizaci√≥n

### 1. Transfer Learning (No Entrenar Desde Cero)

```python
import segmentation_models_pytorch as smp

# Usar encoder pre-entrenado (ImageNet)
model = smp.Unet(
    encoder_name="resnet34",        # Ligero y efectivo
    encoder_weights="imagenet",     # Pre-trained
    in_channels=3,
    classes=1
)

# Solo fine-tune el decoder (m√°s r√°pido)
for param in model.encoder.parameters():
    param.requires_grad = False  # Congelar encoder
```

**Ahorra:**
- 80% de tiempo de entrenamiento
- 70% de recursos GPU
- Mejores resultados con menos datos
