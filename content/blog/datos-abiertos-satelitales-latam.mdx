## üí° Consejos para Proyectos en LATAM

### Desaf√≠o: Cobertura de Nubes

Regiones tropicales (Amazon√≠a, Centroam√©rica) tienen nubes constantes.

**Soluciones:**
1. Usar **Sentinel-1 (radar)**: Penetra nubes
2. **Composites temporales**: Mediana de 3-6 meses
3. **Sentinel-2 Cloud Probability**: Filtrar con umbral <20%

```python
# Composite libre de nubes
def maskS2clouds(image):
    qa = image.select('QA60')
    cloudBitMask = 1 << 10
    cirrusBitMask = 1 << 11
    mask = qa.bitwiseAnd(cloudBitMask).eq(0).And(
           qa.bitwiseAnd(cirrusBitMask).eq(0))
    return image.updateMask(mask)

composite = collection.map(maskS2clouds).median()
```

### Optimizar Descargas

- **Usar tiles procesados**: Sentinel-2 L2A en vez de L1C
- **Definir ROI precisa**: No descargar escenas completas si no es necesario
